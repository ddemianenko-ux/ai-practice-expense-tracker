# CLI Калькулятор витрат

CLI-інструмент для відстеження особистих витрат, створений за допомогою AI-асистентів.

## Використані AI-інструменти
[cite_start]Відповідно до загальних вимог[cite: 6], використано наступні інструменти:
* **Gemini**: Генерація базової структури проекту, реалізація логіки агрегації даних, робота з JSON та допомога з налаштуванням середовища виконання.
* **GitHub Copilot**: Допомога в написанні парсера аргументів командного рядка (`argparse`).

## [cite_start]Інструкція запуску [cite: 21]

Переконайтеся, що у вас встановлено Python (і додано до PATH). Усі залежності стандартні.

1. **Додавання витрати:**
   ```bash
   python src/expense_tracker.py add --category "Продукти" --amount 500 --desc "Сільпо"

2. **Перегляд усіх витрат:**
   ```Bash
   python src/expense_tracker.py list
3. **Перегляд витрат за конкретною категорією:**
   ```Bash
   python src/expense_tracker.py list --category "Продукти"
4. **Загальна статистика:**
   ```Bash
   python src/expense_tracker.py summary
