# CLI Калькулятор витрат

CLI-інструмент для відстеження особистих витрат.

## Використані AI-інструменти
* **ChatGPT**: Генерація базової структури, функцій агрегації даних та логіки роботи з JSON.
* **GitHub Copilot**: Написання парсера аргументів командного рядка (`argparse`).
